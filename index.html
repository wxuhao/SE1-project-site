Project description and background

In front-end web development, itâ€™s often the case that features that are supported on one browser are not supported for certain others. Because of this, web developers must test their websites on different web browsers multiple times. This process becomes tedious for larger websites, especially since developers need to switch between different web browsers, refreshing each multiple times throughout the design process. To make this process more efficient, our group will be designing and prototyping a cross-platform solution plugin, X-Browser, for the open-source text editor Atom. This plugin will allow front-end developers to simultaneously view their project on multiple supported browsers in real-time. We believe that X-Browser will greatly increase productivity in the web design process.
