X-Browser Team SRS Prototype Bibliography
This is the homepage for team X-browser-1.
This semester (Fall 2020), we will be designing and prototyping the X-browser, an Atom plugin that will allow front-end developers to view their website from multiple types of browsers at the same time, in real time. Using the X-browser, web developers will be able to see the changes they've made on Chrome, and simultaneously make sure that those changes are compatible with Internet Explorer.
